# Storytelling Content Prompt

## Purpose
To create relatable, high-engagement content using personal experiences.

## Prompt
Write a short-form story using this structure:
- “I used to…”
- What went wrong
- What I realized
- The lesson learned

Topic: [insert topic]

## Example
"I used to think loyalty meant staying through anything… until I realized I was just tolerating disrespect. Now I choose peace over proving myself."
