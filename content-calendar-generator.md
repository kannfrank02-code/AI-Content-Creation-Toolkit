# Weekly Content Calendar Generator

## Purpose
To create a consistent posting schedule across platforms.

## Prompt
Create a 7-day content plan for TikTok, Instagram, YouTube, and Facebook based on the topic: [insert topic].

Include:
- Daily themes
- Post types
- Content ideas

## Example
Output:
Monday: Motivation Reel  
Tuesday: Tarot Insight  
Wednesday: Storytime  
Thursday: YouTube Video  
Friday: PSA Clip  
Saturday: Live Session  
Sunday: Reflection Post
